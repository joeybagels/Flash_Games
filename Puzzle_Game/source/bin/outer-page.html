Marvel Ad Harness

This is the Ad Harness page for the Marvel website. The enclosed iframe can be placed anywhere in any page and the harness will ensure that the ads are run at the appropriate time during gameplay.


How It Works

The outer-page.html is intended to be the containing page of the game. This can represent any page in the Marvel website, OR external website page on other websites. The iframe code inside the out-page.html can be passed to outside websites provided that the crossdomain.xml file allows for report access from said site.

What You're Seeing

The RED pages are outside Ads that are being placed over the game. The BLUE rectangles are the sample game that is used to ensure that everything is working properly.
