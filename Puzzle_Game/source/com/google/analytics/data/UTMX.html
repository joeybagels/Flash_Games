Urchin Tracking Module Cookie X. The Website Optimizer cookie. This cookie is used by Website Optimizer and only set when the Website Optimizer tracking code is installed and correctly configured for your pages. See the Google Website Optimizer Help Center for details. expiration: 2 years from set/update. note: - ALPO value - document.location.search
